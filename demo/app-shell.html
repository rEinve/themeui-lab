App Shell Pattern

Top bar and off-canvas navigation pattern using the existing app shell classes.

Main Region

On small screens, the navigation collapses into an off-canvas drawer. On larger screens, it returns to an inline top bar.

This is a pattern-level feature because it controls page shell, major regions, and navigation behavior.
