Components

Core components are reusable UI pieces that style their own interior and state only.

Canonical component classes are .button, .card, .field-label, .field-hint, .field-error, .field-success, .field-warning, .input, .select, .textarea, .radio, .checkbox, .switch, .tag, .divider, and .pagination. The legacy .btn alias remains temporarily for compatibility only.
