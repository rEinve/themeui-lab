themeui-lab documentation

Docs directory

All active documentation now lives inside the docs/ folder. The docs are arranged bottom-up by system layer: foundations first, then icons, primitives, components, patterns, utilities, and finally composition and rules. Each section keeps intro and examples separate, and the sidebar links to the actual item pages.
