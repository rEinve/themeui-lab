Primitives

Primitives handle page shell and content flow layout: structure, rhythm, and grouping.

Canonical usage: .container, .section, .grid-12, .stack, .row, .cluster. Legacy aliases like .frame, .section-shell, .page-grid, .content-grid, .stack-tight, and .stack-loose remain for compatibility only.

Stack Example

Use .stack to create consistent vertical rhythm between related blocks.

Overview

First item in the vertical flow.

Details

Second item with the same spacing rhythm.

<div class="stack">
  <article class="card">
    <h3 class="card__title">Overview</h3>
    <p class="card__text">First item in the vertical flow.</p>
  </article>
  <article class="card">
    <h3 class="card__title">Details</h3>
    <p class="card__text">Second item with the same spacing rhythm.</p>
  </article>
</div>
