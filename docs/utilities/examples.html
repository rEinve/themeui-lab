Utilities Examples

Utilities are small, explicit helpers that refine an existing primitive. They do not replace layout primitives or components.

Start with a primitive such as .stack, .row, .cluster, or .grid-12. Then add one utility for the adjustment you need. If you are trying to center a div, first decide whether you want to center the block itself, center the children inside a flow, or center content on both axes inside a container.

Block and Text Alignment

Use these when the centering target is the element itself or just the text inside it.

  • .margin-inline-auto
  • .text-left
  • .text-center
  • .text-right

Flex and Grid Alignment

Use these on primitives that already define layout behavior.

  • .items-top, .items-middle, .items-bottom
  • .justify-start, .justify-center, .justify-end, .justify-between
  • .self-top, .self-middle, .self-bottom
  • .item-grow
  • .place-center

Spacing and Surface Helpers

Use these for small adjustments, not for page composition.

  • .margin-xxl, .margin-xl, .margin-tb-m, .margin-b-m, .margin-t-m
  • .padding-xxl, .padding-t-xxl, .padding-b-xxl, .padding-b-m, .padding-l, .padding-b-l, .padding-xl
  • .border-t, .border-b, .surface
  • .border-rs, .border-rm, .border-rl, .border-rxl
  • .aspect-standard

Center One Div Horizontally

Use .margin-inline-auto when the block itself should sit in the horizontal center.

The block must be narrower than the available width, so pair the utility with a constrained width or max-width. This is the clearest answer when you want to center a single card or panel on the x-axis.

Centered Panel

The panel itself is centered, not just its text.

<div class="card utility-sample margin-inline-auto">
  <h3 class="card__title">Centered Panel</h3>
  <p class="card__text">The panel itself is centered, not just its text.</p>
</div>

Center Children on the Horizontal Axis in a Stack

Use .items-middle on .stack when a vertical flow should center its child blocks horizontally.

Card A

Card B

<div class="stack stack--sm items-middle">
  <div class="card utility-sample utility-sample--compact">Card A</div>
  <div class="card utility-sample utility-sample--compact">Card B</div>
</div>

Center Items Horizontally in a Row

Use .justify-center on .row when the row content should gather in the middle.

<div class="row justify-center items-middle">
  <button class="button" type="button">Back</button>
  <button class="button" type="button">Preview</button>
  <button class="button" type="button">Publish</button>
</div>

Center a Div on Both Axes

Use a flex primitive with both .justify-center and .items-middle when the child should sit in the middle of a taller area.

Centered in Area

This uses available utilities for true two-axis centering.

<div class="row justify-center items-middle" style="min-height: 12rem;">
  <div class="card utility-sample">
    <h3 class="card__title">Centered in Area</h3>
    <p class="card__text">This uses available utilities for true two-axis centering.</p>
  </div>
</div>

What .place-center Actually Does

Use .place-center on .grid-12 when items should center inside their own grid areas.

.place-center sets place-items: center. It is useful for centering content inside grid cells. It is not the primary tool for centering one standalone panel in a section.

Do

Pick the smallest utility that matches the layout adjustment you need, and keep the main structure in primitives.

Don't

Use utilities to replace page composition, container width, section spacing, or component-specific structure.
